## About The Project
This is a simple react front end to interact with the niko smart contracts on ropsten.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ThalKod/niko_frontend.git
   ```
2. Install NPM packages
   ```sh
   cd niko_frontend
   yarn install
   ```
3. Start the app
   ```sh
   yarn start
   ```
